## Overview

This repository contains the source code of my professional website.

It serves as a centralized access point to my work in:

- Data Science  
- Machine Learning  
- Quantitative Modeling  
- Applied Analytics  

The objective is clarity, structure and direct access for technical evaluation.

---

## Architecture

The platform follows a static architecture approach:

- No heavy frameworks  
- No external UI libraries  
- No unnecessary dependencies  

The focus is performance, control and simplicity.

---

## Technical Stack

- Semantic HTML5  
- Modular CSS3  
- Vanilla JavaScript  
- Optimized static assets  
- GitHub Pages deployment
